

# Creative Tech XYZ — Futuristic Technology Education Platform

## Overview
A high-end, dark-themed futuristic website for Creative Tech XYZ — a technology education and innovation company. The site will feel like a next-generation AI startup platform with stunning animations, glowing gradients, and immersive visuals.

## Design System
- **Theme**: Dark futuristic with deep navy/black backgrounds
- **Accents**: Glowing cyan, electric blue, and purple gradients
- **Typography**: Clean, modern sans-serif with large bold headlines
- **Effects**: Glassmorphism cards, glowing borders, particle animations, smooth scroll transitions

## Pages & Sections

### 1. Home Page (Hero + Key Sections)
- **Hero Section**: Full-screen with animated particle/neural network background, headline "Building the Intelligent Generation of Tomorrow", two CTA buttons (Explore Programs, Join the Community)
- **Key Stats Strip**: Animated counters for impact metrics
- **Core Values**: Icon cards with hover glow effects (Innovation, Creativity, Knowledge Sharing, etc.)
- **Learning Ecosystem Preview**: Visual overview of what the platform offers
- **CTA Banner**: Inspiring call-to-action to join

### 2. About Page
- Company introduction and philosophy
- Timeline or visual storytelling of the company's journey
- Core values displayed with animated icons

### 3. Vision & Mission Page
- Vision statement with bold typography and ambient animations
- Mission points displayed as an interactive list/grid
- "Building the future" visual narrative

### 4. Programs Page
- Program cards for each area: AI Fundamentals, Technology Literacy, Innovation Thinking, Digital Productivity, Automation Tools, Entrepreneurship Mindset, Problem Solving, Future Tech Skills
- Each card with icon, description, and hover animation
- No pricing displayed anywhere

### 5. Skills & Technologies Page
- Visual grid/showcase of skills taught
- Animated tech stack icons
- Progress-style visual elements

### 6. Innovation Lab Page
- Virtual lab concept with futuristic visuals
- Features: experiment with ideas, build solutions, develop projects, think like founders
- Interactive cards with glow effects

### 7. Community Page
- Community vision: students, creators, tech enthusiasts, future entrepreneurs
- Collaboration features showcase
- Join community CTA

### 8. Founder Message Page
- Ahmad Syed's story and message
- Professional layout with founder quote
- Vision statement with ambient background

### 9. Contact Page
- Modern contact form (Name, Email, Message) with glowing input fields
- Animated submit button
- Company info sidebar

## Navigation
- Sticky top navbar with glass effect and glow on scroll
- Smooth scroll between sections on home page
- Mobile-responsive hamburger menu with slide-in animation

## Animations & Effects
- Animated particle/dot grid background on hero (CSS-based)
- Fade-in and slide-up animations on scroll for all sections
- Glowing gradient borders on cards and buttons
- Hover scale and glow effects on interactive elements
- Smooth page transitions
- Floating geometric shapes as decorative elements
- Typing animation on key headlines

## Technical Notes
- All frontend, no backend needed
- Contact form will show a success toast (no actual sending)
- Responsive design for mobile, tablet, and desktop
- Uses existing shadcn/ui components styled with the dark futuristic theme
